DevEtym · 촌스럽고 가시적인 버전v2

v2 방향 전환: v1의 "조용한 터미널 위의 사전"은 홈스크린 경쟁에서 너무 점잖다. 이번엔 90–2000년대 한국 문방구·참고서 표지·학원 간판 무드 — 채도 높은 원색 풀배경, 두꺼운 검정 외곽선, 큼지막한 한글 볼드 타이포. "세련됨"을 버리고 0.3초 인식 + 한국어 사용자 직격만 남긴다.

빅 타이포 “어원”

카퍼 풀배경 + 흰색 한글 ExtraBold "어원" 두 글자 + 테두리 이중선. 참고서 표지.

핵심 시각 요소
한글 "어원" 두 글자가 아이콘 면적의 70%를 차지. 하단 작은 mono 라틴 태그라인.
왜 이 브리프에 맞는가
한국어 사용자에게 "어원"은 단어 그 자체가 카테고리다. 이미지 해석 단계를 건너뛰고 글자로 직격한다. 두꺼운 흰 이중 테두리는 90년대 참고서·학원 간판의 관용 장치로, 촌스러움의 "정직한 표지" 정서를 즉시 소환한다.
리스크
한국어 모르면 기능 0. 타이포 아이콘은 App Store 썸네일 경쟁에서 "정보는 많으나 이미지적 임팩트는 없다"는 평가를 받을 수 있음.
28px 실루엣 앵커
"어원" 두 글자 자체. 한글 구조의 네모 블록이 작아져도 한글로 인지됨 + 카퍼 풀배경의 컬러 앵커.

책 + 돋보기 · 클립아트

딥블루 위에 카퍼 책과 라임 돋보기. 아이콘 모음 CD-ROM에서 튀어나온 듯한 정직한 클립아트.

핵심 시각 요소
펼쳐진 사전 책(크림색 페이지) 위에 큰 라임 돋보기가 겹쳐짐. 전체에 두꺼운 검정 아웃라인.
왜 이 브리프에 맞는가
"사전+검색"이라는 기능을 글자 없이 그림만으로 전달. 어휘·언어장벽 없음. 풀배경 원색 + 두꺼운 외곽선은 홈스크린 어떤 배경지에서도 식별된다. 의도적 촌스러움이 오히려 "친근하고 쉬운 앱" 인상을 준다.
리스크
"개발자용"이라는 정보 0. 범용 영어사전·어린이 사전과 구분 불가. 타이포 추가로 보정 필요할 수 있음.
28px 실루엣 앵커
카퍼 책 박스 + 라임 돋보기 원 — 색-형태 듀얼 앵커. 작아져도 "책 옆 동그란 것"으로 읽힘.

네온 간판 { e }

v1의 {e}를 PC방·학원 간판으로. 검정 배경 + 형광 라임 중괄호 + 형광 오렌지 볼드 e.

핵심 시각 요소
두꺼운 고딕 중괄호가 네온관처럼 발광, 가운데 볼드 sans-serif `e`가 대비색으로 꽂힌다. 이중 외곽선 프레임이 간판 테두리 역할.
왜 이 브리프에 맞는가
v1 {e}의 인식 구조(코드⊃어원)는 유지하되, 톤을 "세련된 마크"에서 "간판"으로 바꿔 홈스크린 어둠 속에서 발광한다. 한국 사용자가 90–2000년대 학원 간판에서 본 "{컴퓨터}" 로고의 시각 문법을 그대로 소환 — 즉시 친숙함 + 뚜렷한 식별력.
리스크
형광색은 iOS 기본 아이콘들 옆에서 "조잡함"으로 읽힐 수도 있음. 좋아하는 사람과 싫어하는 사람이 극명히 갈리는 스타일.
28px 실루엣 앵커
검정 배경 + 라임 중괄호 두 곡선 — 형태 + 고채도 색의 이중 앵커로 축소 강함.

고전 사전 표지

녹색 가죽 표지의 전문사전 표지 복각. "DEV ETYM / 개발어원 / 사전" 3단 구성.

핵심 시각 요소
olive 풀배경 + 크림색 이중 테두리 + 영문 serif 타이틀 + 한글 볼드 부제 + 하단 "사전" 라벨 박스. 도서관 장서 책등 스타일.
왜 이 브리프에 맞는가
"이건 진짜 사전이에요"를 텍스트로 세 번 반복한다. "DEV ETYM" → "개발어원" → "사전". 글자 크기가 단계적이라 사이즈별로 살아남는 글자가 달라짐 — 28px에서도 "개발어원" 네 글자는 확실히 읽힌다. 고전 사전 장정을 그대로 빌려와서 "신뢰할 수 있는 레퍼런스" 정서를 즉발.
리스크
영문 부분이 28px에서 소실. 타이포 밀도가 높아 홈스크린에서 "글자 아이콘" 그룹에 묻힐 수 있음. 한국어 모르면 접근성 급락.
28px 실루엣 앵커
olive 풀배경 + "개발 / 어원" 두 줄 한글 덩어리 — 텍스트 덩어리의 매스감.

책 + 한글 ‘어’ 스티커

크림 배경, 두꺼운 검정 외곽선의 펼친 책 위에 삐뚤어진 오렌지 스티커 + 한글 ‘어’.

핵심 시각 요소
살짝 기울어진 펼친 책, 왼쪽 페이지 위에 원형 오렌지 스티커(8도 회전)에 볼드 '어' 한 글자. 배경엔 문방구 점 패턴 소량.
왜 이 브리프에 맞는가
책(사전) + 한글 '어'(어원)의 두 시그널을 모두 유지하되, 기하학적 완벽성 대신 "손으로 대충 붙인 스티커" 정서를 더한다. 이 정서가 "친근한 학습 도구" 인상을 즉시 만든다. 28px 축소 시 책이 사라져도 오렌지 원형 + 검정 '어'가 아이콘 중앙을 지배하므로 인식 유지.
리스크
요소가 세 층(패턴·책·스티커)이라 작은 사이즈에서 복잡도 과잉. 기울기·점 패턴이 "정돈되지 않음"으로 읽힐 위험.
28px 실루엣 앵커
중앙 오렌지 원 + 검정 '어'. 책은 과감히 생략하고 스티커만 크게 살리는 대체 버전 포함(위 28px).

추천 순위 · v2

  1. 책 + 한글 ‘어’ 스티커 card 05
    v2의 지향("촌스러움 + 가시성 + 친근함")을 가장 균형 있게 달성. 책(범용 사전 시그널) + '어'(한국어 직격) 두 신호가 공존하며, 삐뚤어진 스티커 처리로 "정직한 학습 도구" 정서 추가. 28px에서도 스티커만 남아 식별 유지.
  2. 빅 타이포 “어원” card 01
    한국 사용자 직격 효율 1위. 이미지 해석 단계를 건너뛰고 글자로 꽂힌다. 참고서 표지의 권위 정서를 즉발. 타이포 아이콘의 "이미지적 임팩트 부족"만 감수할 수 있다면 최상위.
  3. 네온 간판 { e } card 03
    v1 5번을 아주 좋아했다면 이 버전이 정확한 연결 고리. 형태 논리는 그대로, 톤만 학원 간판 네온으로. 호불호 강하지만 "촌스럽고 눈에 띄게"라는 브리프에 가장 직접적으로 응답.
  4. 책 + 돋보기 · 클립아트 card 02
    가장 "안전한 촌스러움". 범용 사전·검색 앱으로 즉시 분류되지만 "개발자용"이라는 특화 정보 0. 소구력은 높으나 차별화는 낮음.
  5. 고전 사전 표지 card 04
    텍스트 밀도가 가장 높아 정보는 풍부하지만 28px에서 가장 많은 요소가 무너짐. 가독성 기준으로 후순위. App Store 썸네일 전용이라면 매력적이나 홈스크린 아이콘으로는 무거움.
다음 단계. v1·v2 합쳐 10개 중에서 최종 픽이 결정되면 1024×1024 PNG + iOS 아이콘셋 전체 사이즈 출력 가능. 모든 SVG가 viewBox 1024×1024로 통일되어 있어 무손실 스케일링. v2는 풀컬러 배경이 많아 라이트/다크 듀얼 톤 대신 단일 톤으로 가는 것이 정체성상 맞다.
