DevEtym — a symbol-first approach

브리프를 다시 읽으면 개발 용어의 "어원"을 파헤치는 사전이다. "책 + 코드 괄호", "돋보기 + 글자" 같은 조합은 이 장르에서 가장 흔한 1세대 발상이고, 결과물이 AI가 뱉어낸 듯한 아이콘으로 수렴한다.

이번엔 장면 대신 심볼로 접근한다. 각 후보는 DevEtym이 다루는 세계관 — 역사언어학, 타이포그래피, 수학/CS, 동아시아 주석 관례 — 에서 실제로 쓰이는 표기법을 마크로 끌어올린 결과다. 책도, 돋보기도, tym. 같은 설명도 없다. 기호 하나로 의미가 닫힌다.

01Proto-form Asterisk

역사언어학에서 *는 "문헌에 남지 않았지만 재구된 조상어"를 표시한다. *dewós, *ph₂tḗr처럼. 어원학이 자기 자신을 표기하는 방식을 그대로 마크로.

세 개 선이 교차하는 기하학적 6각 별표. 15° 회전으로 미세한 긴장. 28px에서도 형태가 무너지지 않는 실루엣. 아는 사람에게는 도메인 코드로 읽힌다.

02de.

라틴어 접두사 de-는 "에서, ~로부터"의 뜻 — 어원의 본질. 동시에 DevEtym의 첫 두 글자. 은유를 걷어내고, 이름으로 자신을 증명하는 워드마크.

헤비웨이트 소문자 + 타이트한 커닝 + 약호 마침표. Linear·Vercel·Arc가 쓰는 방식이다. 설명하지 않고 존재한다. 홈스크린에서 가장 빨리 눈에 박힐 후보.

03/…/

슬래시 쌍 /.../은 사전에서 음성기호이고, 유닉스에서는 경로다. 같은 기호가 두 세계를 잇는다 — 그게 DevEtym의 일이다.

가운데 빈 공간 + 생략부호 3점은 "여기 어떤 단어든 들어올 수 있다"는 초대. 시스템이 확장될 때 /dev/, /etym/ 같은 변주도 자연스럽다.

GreekCS
04λ

람다의 이름은 페니키아 문자 𐤋(lamed)에서 그리스어 λ, 다시 Church의 λ-calculus, 그리고 모든 함수형 언어로. 한 글자에 3천 년의 어원이 담겨 있다. 아이콘이 곧 어원 사례.

폰트가 아니라 두 획으로 직접 구성 — 시스템 간 렌더링 일관성. AWS Lambda·Haskell·Lisp에 이미 자리 잡았지만, 다크 배경 + 라임 조합은 이 셋 어디에도 없다. 차별화 가능.

05※

한국·일본 출판물에서 ※는 "참고", "주석을 붙이니 주의하라"의 기호. 한국 개발자 타겟 앱에 로컬 레퍼런스 마크를 박는 건 브랜드 정체성 선언이다.

6획 중심 별표 + 4개 코너 닷. 국제 관객은 "흥미로운 기하학적 기호"로 인식하고, 한국/일본 사용자에게는 즉각 주석/사전으로 해독된다. 문화적 공명이 가장 강한 후보.

#0a0a0a 배경 / #c8f060 accent / 1024×1024 뷰박스 기준. 픽한 방향으로 디테일 다듬어 PNG 뽑음.
