개발어원 · 사전 · 최종안

v2 Card 04 기반. 영문 "DEV ETYM"을 제거하고 한글 "개발어원"에 모든 시선을 몰았다. 색상은 #2E5D3A(딥 그린) — 네이버 사전·고전 영한사전 표지의 계열로, 한국 사용자에게 "사전·학습 도구" 카테고리를 이미지 해석 없이 즉시 발사한다. 크림색 이중 테두리 + 상·하 장식 라인 + "사전" 라벨 박스는 고전 사전 장정의 관용 장치.

사이즈별 가독성

iOS 아이콘 실사용 사이즈. 180px는 앱 스토어, 120px는 iPad 홈, 60–80px는 iPhone 홈, 40px는 설정·스포트라이트, 28px는 알림·검색 결과. 모든 사이즈에서 "개발어원" 네 글자가 살아남도록 설계 — 28px에서도 한글 덩어리는 유지된다.

디자인 스펙

배경
#2E5D3A (딥 그린). 네이버 사전·고전 영한사전 표지 계열. 크림과의 WCAG 대비비 6.46:1 (AA 통과).
타이포
Apple SD Gothic Neo / Noto Sans CJK KR / Pretendard 900(Black). "개발 / 어원" 2행 배치, letter-spacing 24로 한글 정사각 밸런스 유지. "사전" 라벨은 Black 60pt, letter-spacing 14.
컬러 요소
메인 타이포 + 테두리 + 장식 라인 모두 #F7E8D0(크림). "사전" 박스만 반전: 크림 배경 + 딥 그린 글자.
프레임 장식
이중 테두리(외곽 18px, 내부 5px 크림) + 상·하 장식 라인 2쌍 — 고전 사전 표지 장정의 관용 형태.
사이즈 대응
120px↑ 완전형 / 60–80px 테두리만 / 40px↓ 타이포만. 28px에서도 딥 그린+크림 대비와 한글 덩어리로 식별.
라이트·다크 공용
iOS 18 기본 아이콘은 컬러 고정. 라이트/다크 두 배경 모두에서 딥 그린의 저명도가 경계 역할을 해 자동으로 대비 확보.
팔레트 확장 근거
앱 기존 accent(#3F7A00)과 같은 계열(그린)이지만 더 짙고 블루 쪽으로 기울어, 아이콘을 앱 내부와 구별되는 "표지"로 자리매김함. 즉, 앱 = 책 내용, 아이콘 = 책 표지라는 위계가 자연스럽게 형성.
