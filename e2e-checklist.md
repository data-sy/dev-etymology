# E2E 통합 테스트 체크리스트

> 시뮬레이터에서 수동 검증. 각 항목을 직접 확인 후 체크.
> 문제 발견 시 증상을 옆에 메모하고, `fix/<area>` 브랜치로 수정.

---

## 1. 검색 (Search)

### 번들 DB 검색
- [x] `mutex` 검색 → 즉시 결과 표시 (로딩 없음)
- [x] 카테고리 태그 "동시성 · Concurrency" 배지 표시
- [x] AI 생성 뱃지 **표시되지 않음** (source: bundle)
- [ ] 어원 블록 (좌측 accent 보더) 정상 표시
- [ ] 작명 이유 본문 스크롤 가능

### AI 폴백 검색
- [x] `kubernetes` 검색 → 로딩 화면 ("어원을 분석하는 중...") → AI 결과 표시
- [x] 카테고리 태그 배지 표시
- [x] AI 생성 뱃지 (✦ AI 생성) 표시
- [ ] 두 번째 `kubernetes` 검색 → 캐시 히트 (로딩 없이 즉시 표시)

### 자동완성
- [ ] `mu` 입력 → 자동완성 드롭다운에 `mutex` 등 표시
- [ ] `se` 입력 → `semaphore`, `serialization` 등 표시
- [ ] 자동완성 항목 탭 → DetailView 이동
- [ ] 300ms 디바운스 동작 확인 (빠르게 입력 시 요청 한 번만)

### 에러 플로우
- [ ] 비개발 용어 (`사과`) 검색 → "개발 용어를 검색해주세요" 안내
- [ ] 오타 (`mutix`) 검색 → "{추천 용어}을(를) 찾으셨나요?" → 추천 탭 시 재검색 (push가 아닌 replace)
- [ ] 빈 문자열 (공백만) 입력 후 엔터 → 아무 동작 없음

## 2. 상세 화면 (Detail)

### 레이아웃
- [ ] 용어명 large title (DM Serif Display 폰트)
- [ ] 한 줄 요약
- [ ] 어원 블록 (좌측 accent 보더, DM Mono 언어명, 설명 텍스트)
- [ ] 작명 이유 본문 — 긴 텍스트 시 ScrollView 스크롤

### 액션
- [ ] 북마크 버튼 탭 → 아이콘 상태 변경 (토글)
- [ ] 공유 버튼 탭 → ShareLink 시트 표시, "{keyword}\n\n{summary}\n\n— DevEtym" 형식
- [ ] 하단 "이 설명이 잘못됐나요? 오류 제보하기" → mailto 링크 열림 (제목·본문 자동 채움)

---

## 3. 북마크 (Bookmark)

- [ ] 북마크된 용어가 북마크 탭에 표시
- [ ] 각 항목에 keyword + summary 한 줄 미리보기
- [ ] 항목 탭 → DetailView 이동
- [ ] 스와이프 삭제 → 목록에서 즉시 제거
- [ ] 빈 상태 → "아직 저장한 용어가 없어요" 안내 문구

---

## 4. 히스토리 (History)

- [ ] 최근 검색 기록 표시
- [ ] 상대 시간 표시 ("방금 전", "1시간 전", "어제", "3일 전" 등)
- [ ] 항목 탭 → DetailView 이동
- [ ] 스와이프 삭제 → 해당 항목만 제거
- [ ] "전체 삭제" 버튼 → 모든 히스토리 제거
- [ ] 검색 탭의 최근 검색 칩과 동기화 (히스토리 삭제 후 칩도 갱신)

- [ ] "시작하기" 버튼 → 메인 화면 진입

## 6. 디자인 시스템

### 다크모드
- [ ] 앱 전체 다크모드 기본 렌더링 정상
- [ ] 시뮬레이터 Settings → 외관 → 라이트로 전환 → 전 화면 정상 (깨지는 UI 없음)
- [ ] 다시 다크로 전환 → 정상
- [ ] 하드코딩된 `.white`/`.black` 텍스트 없음 (컬러 대비 깨지는 곳 없음)

### 폰트
- [ ] 용어명 large title → DM Serif Display
- [ ] 코드/라벨/칩/탭라벨 → DM Mono
- [ ] 본문 → DM Sans
- [ ] Dynamic Type: 시뮬레이터 Settings → 접근성 → 큰 텍스트 → 레이아웃 깨지지 않음

### 컬러 팔레트

- [ ] accentAI (#f0a060) — AI 생성 뱃지
- [ ] surface/surface2 — 카드/입력창 배경 구분
